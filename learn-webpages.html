Learn How This Website Works

This page breaks down the three main pieces that power this site. Once you understand these, you can customize everything for your own projects.

HTML = Structure

HTML (HyperText Markup Language) is the skeleton of your page. It defines headings, paragraphs, links, images, and the overall layout.

In this project, each page is a separate .html file: index.html, my-code.html, learn-webpages.html, and publish-github.html.

CSS = Style

CSS (Cascading Style Sheets) controls how everything looks: colors, fonts, spacing, and layout.

We place all styles in one file, styles.css, and link it in each HTML page so the design stays consistent.

JavaScript = Interactivity

JavaScript adds behavior and interactivity to your site. It can respond to clicks, scroll events, and update the page without reloading.

This site uses script.js to: show a Back to Top button, smoothly scroll up, and highlight the active page in the navigation.
