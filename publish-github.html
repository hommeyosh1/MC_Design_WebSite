Publish Your Website for Free

GitHub Pages lets you host static websites (HTML, CSS, JS) for free. It’s perfect for portfolios, small projects, and learning how deployment works.

  1. Create a GitHub account
    Go to github.com and sign up if you don’t already have an account.
  2. Create a repository
    Click New repository, give it a name (for example, mc-desing-webpages), and make it public.
  3. Upload your files
    Add your index.html, other HTML files, styles.css, and script.js to the repository.
  4. Open Settings → Pages
    In the repository, go to Settings, scroll to Pages, and choose the branch (usually main) and folder (/root).
  5. Enable GitHub Pages
    Click Save. GitHub will build your site. This may take a minute.
  6. Copy your website link
    After it’s done, you’ll see a URL like https://your-username.github.io/your-repo-name/. Share this link so others can view your site.

Why Use GitHub Pages?

GitHub Pages is free, easy to set up, and works directly with your code. It’s a great way to:

  • Showcase your projects and portfolio
  • Practice real-world deployment
  • Share live demos with friends, classmates, or clients

Once you know this workflow, you can publish any static site you build — including this MC Desing Webpages project.
